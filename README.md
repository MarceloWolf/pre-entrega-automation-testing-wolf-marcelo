# Pre-entrega Automation Testing - SauceDemo

Este proyecto contiene una suite de pruebas automatizadas para la plataforma SauceDemo utilizando **Python**, **Selenium** y **Pytest**.

## Estructura del Proyecto
- `tests/`: Contiene los casos de prueba (Flujo completo de compra).
- `utils/`: Constantes y funciones auxiliares.
- `reports/`: Reportes de ejecución en formato HTML.
- `requirements.txt`: Librerías necesarias para el proyecto.

## Instalación
1. Clonar el repositorio.
2. Instalar dependencias:
   ```bash
   pip install -r requirements.txt
